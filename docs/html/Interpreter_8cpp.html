The function Initialize the Lexer (Tokenizer), and load & execute each statement in order that they are loaded.

Everything is loaded and executed in sequence. The interpreter will raise any kind while executing the expression. It will not search by any kind of error before start execution.

Parameters
argcThe default argument received by C++ applications
argvThe array containing two files. The first element is the file.tl (program) and the second one is the token file english.yml
Warning
Both file references are required. Otherwise will return an error.
